# mi-primer-repo

Repositorio de prueba para aprender el flujo de trabajo con Git y GitHub.

## Descripción

Este proyecto sirve para practicar:

- Crear ramas
- Hacer commits
- Abrir Pull Requests

## Cómo empezar

1. Clona el repositorio
2. Crea una rama nueva
3. Haz tus cambios
4. Abre un Pull Request
